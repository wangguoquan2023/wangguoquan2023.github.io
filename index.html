概述

欢迎来到 Hexo 文档。如果您在使用 Hexao 时遇到任何问题，请查看故障排除指南，在 GitHub 上提出问题或在 Google 群组上发起主题。

什么是hexo?

Hexo是一个快速，简单和强大的博客框架。您用Markdown（或其他标记语言）撰写帖子，Hexo在几秒钟内生成具有精美主题的静态文件。

安装

只需几分钟即可设置 Hexo。如果您遇到问题并且无法在此处找到解决方案，请提交 GitHub 问题，我们将提供帮助。

要求

安装 Hexo 非常简单，只需要事先执行以下操作：

nodejs
（应至少为 Node.js 10.13，建议为 12.0 或更高版本）
git
如果您的计算机已经有这些，恭喜！您可以跳到hexo安装步骤。
